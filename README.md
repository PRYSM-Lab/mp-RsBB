Source codes of the RsBB algorithm presented in A. Marousi, V. M. Charitopoulos, 2025. Global and robust optimization for non-convex quadratic programs. arXiv, 2503.07310.

Source codes of the accelerated mp-RsBB based on multi-parametric programming for the article '' Accelerated robust spatial Branch-and-Bound algorithm via multi-parametric programming:Application to pooling problems''
